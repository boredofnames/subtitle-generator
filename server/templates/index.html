Subtitles Generator REST/WebSocket Server

Welcome to the subtitle-generator back end. Learn more about the server here.

REST

  • / (GET)
    • Returns this page
  • /vtt (POST)
    • Expects (JSON): {"url": site_url}
    • Returns (JSON): {"vtt": vtt_data}
    • Returns Error (JSON): {"error": error_name, "cause": error_cause}

WebSockets

  • Expects (JSON): {"url": site_url}
  • Returns (JSON): {"type": "vtt", "data": vtt_data}
