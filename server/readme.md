# setup venv
ensure you're in subtitle-generator/server
```
python3 -m venv ./venv
```
# activate venv
bash
```
source ./venv/bin/activate
```
fish
```
source ./venv/bin/activate.fish
```
# install requirements
```
python3 -m pip install -r requirements.txt
```
# run
```
python3 app.py
```
# deactivate venv
```
deactivate
```
